# Bryce Chriestenson Personal Site

Static GitHub Pages starter for a professional personal homepage.

## Publish

1. Copy the contents of this folder into a GitHub repository.
2. In GitHub, go to `Settings -> Pages`.
3. Choose `Deploy from a branch`, then select the branch and `/root`.

No build step is required.

## Files

- `index.html`: page structure and content
- `styles.css`: dark visual system and responsive layout
- `main.js`: animated WebGL background and smooth scrolling

## Before publishing

- Add a real email link if you want direct contact from the page.
- Review dates, titles, companies, and patent/publication wording.
- Add a headshot or project screenshots if you want the page to feel more personal.
